# 신뢰성 NG 처리 절차서 (Reliability NG Handling Process)
**문서코드**: QM.02.02.07  
**시행일**: 2019-06-01  
**관리 부서**: QC 부서  
**적용 규격**: ISO 9001:2015, IATF 16949:2016  

---

## 1. 목적

신뢰성 시험에서 발생한 NG(부적합) 제품에 대한 처리 절차를 수립하여, 신뢰성 불량의 재발을 방지하고 품질 수준을 지속적으로 향상시킨다.

---

## 2. 적용 범위

M&C Electronics Vietnam 유한회사에서 수행하는 신뢰성 시험(환경 시험, 내구성 시험 등) 중 발생하는 모든 NG에 적용한다.

---

## 3. 책임과 권한

| 부서 | 역할 |
|------|------|
| **QC 부서** | 신뢰성 NG 발견, 격리, 보고, OK 스탬프 관리 |
| **R&D 부서** | 기술적 원인 분석, 개선 조치 수립 |
| **생산 부서** | 개선 조치 실행, 양산 복귀 |
| **경영진** | 보고 수령, 전사 공유 지시 |

---

## 4. 절차 내용

### 4.1 신뢰성 NG 처리 흐름

| 단계 | 담당 | 내용 |
|------|------|------|
| **1. NG 제품 발견** | QC 부서 | 신뢰성 시험 중 NG 제품을 발견한다 |
| **2. 생산 중단** | 생산 / QC | 해당 공정 생산을 즉시 중단한다 |
| **3. NG 제품 식별 및 격리** | QC 부서 | NG 제품을 식별 표시하고 정상 제품과 분리하여 격리한다 |
| **4. 상급자 보고 및 확인** | QC 부서 | 상급자에게 보고하고 상황을 확인한다 |
| **5. 전사 공유** | 경영진 / QC | 신뢰성 NG 발생 사실을 전사에 공유한다 |
| **6. NG 수량 집계** | QC 부서 | 실제 불량 발생 수량을 통계적으로 집계한다 |
| **7. 기술적 개선** | R&D / 생산 | 원인을 분석하고 기술적 개선 방안을 수립·실행한다 |
| **8. 신뢰성 시험용 시료 위탁** | R&D / QC | 개선된 제품으로 신뢰성 시험 시료를 제작하여 위탁 시험을 진행한다 |
| **9. OK 스탬프 관리** | QC 부서 | 검사 합격 제품에 OK 스탬프를 부착하여 품질 관리를 실시한다 |
| **10. 개선 조치 전개** | R&D / 생산 | 개선 조치를 전 공정에 적용하여 전개한다 |
| **11. 창고 입고** | 창고 / QC | 최종 합격 제품을 창고에 입고한다 |
| **12. 유효성 검증** | QC 부서 | 개선 조치의 유효성을 검증한다 |

---

## 5. 핵심 원칙

- 신뢰성 NG 발생 시 **즉각적인 생산 중단**이 원칙이다.
- 개선 조치 없이 생산 재개는 금지한다.
- 동일 유형의 불량이 반복될 경우 에스컬레이션 절차에 따라 상위 단계로 보고한다.
