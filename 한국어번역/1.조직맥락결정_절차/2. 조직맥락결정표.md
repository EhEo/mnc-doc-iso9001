## SWOT 분석표

### 강점 (Điểm mạnh / Strengths — 내부 유리 요인)

| 번호 | 내용 |
|------|------|
| 1 | 회사 경영진이 주요 사업 분야에 대한 풍부한 관리 경험을 보유하고 있음 |
| 2 | 유리한 지리적 위치: 교통 접근이 용이하고, 협력 기업과 노동력이 집중된 지역에 위치 |
| 3 | 신식·현대화된 인프라, 생산에 과학기술 적용 → 제품·서비스 품질 향상 |
| 4 | 인력: 회사와 오랫동안 함께 해 온 경험 있는 직원 확보 |
| 5 | 정보기술 소프트웨어 및 시스템 활용으로 정보 교환이 빠르고 편리하며, 인적 오류 방지 가능 |

### 약점 (Điểm yếu / Weaknesses — 내부 불리 요인)

| 번호 | 내용 |
|------|------|
| 1 | 인력 변동이 많고, 장기 근속 직원 비율이 낮음 |
| 2 | 인원 부족으로 업무가 자주 과중되는 상황 발생 |
| 3 | 언어 및 문화적 차이 존재 |

### 기회 (Cơ hội / Opportunities — 외부 긍정 요인)

| 번호 | 내용 |
|------|------|
| 1 | 사회 생활 수준 향상, 경제 성장, 시장 잠재력 확대 |
| 2 | 정치적 안정; 국가 기관의 기업 발전 지원·투자 정책 |
| 3 | 국제 경제 통합의 심화 |
| 5 | 평균 근로 연령이 낮고 역동적인 노동력 |

### 위협 (Thách thức / Threats — 외부 부정 요인)

| 번호 | 내용 |
|------|------|
| 1 | 경쟁업체 증가로 인한 시장 축소 |
| 2 | 지역 내 타 기업과의 인사(人事) 경쟁 |
| 3 | 공급업체 품질 불안정 |
