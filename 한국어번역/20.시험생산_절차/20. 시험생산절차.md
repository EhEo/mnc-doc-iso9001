## 1. 목적 및 적용 범위

### 1.1 적용 범위

본 절차서는 계획, 생산, 검사, 작성, 개선, 시생산, 양산 전 과정에 적용한다.

### 1.2 목적

양산 이전에 생산 공정에서 자주 발생하는 오류를 발견하고, **NG 및 OK 비율을 명확히 파악**하여 적시에 처리·개선함으로써 양산 품질을 보장한다.

---

## 2. 책임과 권한

| 부서 | 역할 |
|------|------|
| **R&D 부서 (개발 책임자)** | 시생산 계획 수립, 생산 준비, 시생산 총괄, CPK 분석, 보고서 작성, 개선 실행 |
| **생산 부서 (생산 책임자)** | 시생산 수행, 생산 파라미터 기록, 각 단계 생산 보고서 작성 |
| **품질(QC) 부서 (QC 책임자)** | 검사 수행, NG 결과 보고, 각 단계 품질 데이터 취합 |
| **영업·창고 부서** | 검사 완료 제품의 고객 납품 |

## 3. 절차 내용

### 3.1 프로세스 흐름

| 단계 | 담당 | 내용 | 입력 | 출력 |
|------|------|------|------|------|
| **1. 계획 수립** | 개발 책임자 | 고객 요건을 파악하고 자재 구매 및 생산 부서와 협력하여 시생산 계획을 수립한다 | 고객 수량 요건 | 시생산 계획서 |
| **2. 생산 및 검사 준비** | 개발·생산·QC 책임자 | 시생산 계획에 따라 보조 자재를 준비·보관하고, QC 부서에 입고 제품 검사를 요청한다 | 생산 계획서 | 외관 검사표, 시험된 자재 |
| **3. 시생산 수행** | 생산·개발·QC 책임자 | 시생산 계획에 따라 생산을 진행하고, 각 단계의 파라미터 및 조건을 모두 기록한다 | 지그 프레임 확인, 기계 종류, 자재 | 시료 제품 |
| **4. 점검·보고서 작성·오류 분석** | 개발·생산·QC 책임자 | 각 단계 생산 완료 후 생산·QC 부서가 오류를 점검·집계하여 QC 팀장에게 보고한다 | 각 단계 QC 부서 NG 시험 결과 | 이메일, 회의록, 오류 분석 보고서 |
| **5. 단계별 보고서 작성** | 생산·개발·QC 책임자 | 생산 책임자는 각 단계의 사양, 실제 생산 조건, 담당자, 생산 수량 보고서를 작성한다 | NG 분석 결과 | 이메일, 단계별 생산 보고서 |
| **6. 공정 능력 지수(CPK) 측정** | 개발·생산·QC 책임자 | 생산 관련 모든 설비·조건을 교정하고, **1로트(50개)를 연속 생산**하여 CPK를 산출한다. **주의: 제품 치수의 돌발 변동을 방지하기 위해 CPK 측정 중 기계를 재교정하지 않는다** | 생산 공정 관련 설비의 모든 조건·사양 | CPK 평가표, 설비 시험표 |
| **7. 검사 및 납품** | 영업·창고 부서 | 검사 완료된 제품을 고객에게 발송한다 | 시험된 제품 | 납품 명세서 |
| **8. 보고서 작성** | 개발·QC 책임자 | 시생산 후 시생산 보고서를 작성한다 | 시생산 수량 시험 결과 | 품질 시험 보고서, 시생산 완료 보고서 |
| **9. 개선 실행** | 개발 책임자 + 관련 부서 | 고객 피드백을 반영하여 개선을 실행한다 | 고객 피드백 | 회의록, 조치 및 개선 결과 |
| **10. 초기 양산** | 개발 책임자 | 개선·조치 완료 후 양산을 시작한다 | 시생산 보고서 | 생산 보고서, 설비 시험표 |
