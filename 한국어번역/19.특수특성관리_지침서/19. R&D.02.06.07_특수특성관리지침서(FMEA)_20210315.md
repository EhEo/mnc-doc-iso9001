## 1. 목적

제품의 각 제조·생산 공정 단계에서 잠재적 고장 유형 및 그 영향을 사전에 파악하여, 생산 공정 중 및 특히 제품이 고객에게 납품된 이후 해당 고장이 발생하지 않도록 적시에 예방 조치를 취한다.

---

## 2. 적용 범위

- **PFMEA(공정 FMEA)만 적용**한다. 단, 회사가 고객에게 제안하는 설계 변경이 있을 경우 설계 FMEA(DFMEA)도 수행한다.
- **자동차 부품**에 대해 PFMEA를 수행하거나, 고객이 요구할 경우 수행한다.

---

## 3. 정의 및 약어

| 약어/용어 | 정의 |
|-----------|------|
| **FMEA** | 잠재 고장 유형 및 영향 분석 (Potential Failure Mode and Effects Analysis) |
| **PFMEA** | 공정 FMEA (Process Failure Mode and Effects Analysis) |
| **DFMEA** | 설계 FMEA (Design Failure Mode and Effects Analysis) |
| **CFT** | 다기능팀 (Cross-Functional Team) |
| **S (Severity)** | 심각도 (고장의 영향 심각성) |
| **O (Occurrence)** | 발생도 (고장 원인의 발생 가능성) |
| **D (Detection)** | 검출도 (고장의 검출 가능성) |
| **RPN** | 위험 우선순위 지수 (Risk Priority Number = S × O × D) |
| **CC** | 중요 특성 (Critical Characteristic) |
| **SC** | 중요 특성 (Significant Characteristic) |
| **HI** | 고영향 (High Impact) |
| **OS** | 작업자 안전 (Operator Safety) |
| **CTQ/CTP** | 품질 핵심 특성 / 공정 핵심 특성 |
| **SPC** | 통계적 공정 관리 (Statistical Process Control) |

> **FMEA 핵심 원칙**: FMEA는 "사건 발생 후" 대응이 아닌 "사건 발생 전" 예방적 행동이다.

---

## 4. 절차 내용

### 4.1 FMEA 수행 시기

- 제품 설계 및 시료 제작 단계 (공정 설계 단계)
- 「제품 설계 및 개발 절차」에 따른 설계 변경 발생 시

### 4.2 다기능팀(CFT) 구성

- R&D 부서, 금형팀, QC 부서, 관련 생산 부서의 대표자로 구성
- FMEA는 한 명이 아닌 팀 전체의 지식과 경험을 활용하는 것이 원칙
- 회의 주관: R&D 부서 (해당 프로젝트 담당자/부서장)

---

### 4.3 심각도(S) 평가 기준표

| 영향 | 고객에 대한 영향 | 생산에 대한 영향 | 점수 |
|------|----------------|----------------|------|
| 경고 없는 위험 | 사전 경고 없이 제품 안전 사용 또는 정부 법규 미준수에 영향 | 경고 없이 설비 또는 생산 라인에 위험 초래 | **10** |
| 경고 있는 위험 | 사전 경고 후 제품 안전 사용 또는 정부 법규 미준수에 영향 | 경고 있이 설비 또는 생산 라인에 위험 초래 | **9** |
| 매우 높음 | 주요 기능 상실 | 100% 폐기 또는 1시간 초과 수리 | **8** |
| 높음 | 제품 작동 가능하나 기능 저하, 고객 불만족 | 별도 분류 후 폐기 (100% 미만), 30분~1시간 수리 | **7** |
| 보통 | 제품 작동하나 편의 기능 불작동, 고객 불만족 | 분류 후 폐기 (100% 미만), 30분 미만 수리 | **6** |
| 낮음 | 제품 기능 충족하나 성능 저하, 고객 가끔 불만족 | 100% 재작업 또는 생산 라인 외 수리 | **5** |
| 매우 낮음 | 기밀·유격·소음 등 기준 미달, 고객 불만율 75% 초과 | 분류 필요하나 폐기 없음 | **4** |
| 경미 | 기밀·유격·소음 등 기준 미달, 고객 불만율 50% | 폐기 없음, 생산 라인 내 즉시 수리 가능 | **3** |
| 매우 경미 | 기밀·유격·소음 등 기준 미달, 고객 불만율 25% 미만 | 폐기 없음, 생산 스테이션에서 즉시 수리 | **2** |
| 없음 | 영향 없음 | 약간의 불편 또는 고객 불만 없음 | **1** |

---

### 4.4 발생도(O) 평가 기준표

| 발생 가능성 | 불량률 | 점수 |
|------------|--------|------|
| 매우 높음: 지속적으로 발생 | ≥100/1,000 (1건/10건) | **10** |
| 매우 높음 | 50/1,000 (1건/20건) | **9** |
| 높음: 빈번 발생 | 20/1,000 (1건/50건) | **8** |
| 높음 | 10/1,000 (1건/100건) | **7** |
| 보통: 간헐적 발생 | 2/1,000 (1건/500건) | **6** |
| 보통 | 0.5/1,000 (1건/2,000건) | **5** |
| 보통 | 0.1/1,000 (1건/10,000건) | **4** |
| 낮음: 드물게 발생 | 0.01/1,000 (1건/100,000건) | **3** |
| 매우 낮음 | ≤0.001/1,000 (1건/1,000,000건) | **2** |
| 원격: 거의 발생 안 함 | 예방 관리를 통해 고장이 제거됨 | **1** |

---

### 4.5 검출도(D) 평가 기준표

| 검출 가능성 | 기준 | 점수 |
|------------|------|------|
| 거의 불가능 | 검출 불가능 또는 검사 불가 | **10** |
| 매우 낮음 | 간접적 관리 또는 무작위 샘플링만 가능 | **9** |
| 낮음 | 육안 관찰만으로 관리 | **8** |
| 매우 낮음 | 이중 육안 검사만으로 관리 | **7** |
| 낮음 | SPC 모니터링 차트로만 관리 | **6** |
| 보통 | 생산 후 측정 장비 또는 100% Go/No-Go 검사 | **5** |
| 보통~높음 | 다음 공정에서 검출 또는 최초 부품 검사로 검출 | **4** |
| 높음 | 현장에서 즉시 검출, 불합격 부품 자동 배출 | **3** |
| 매우 높음 | 현장 자동 검사 설비로 불량 자동 배출 | **2** |
| 거의 확실 | 설계 또는 생산 중 Poka-Yoke로 고장 발생 자체가 불가능 | **1** |

---

### 4.6 위험 우선순위 지수(RPN) 계산

**RPN = 심각도(S) × 발생도(O) × 검출도(D)**

> **RPN > 100인 경우 반드시 개선 조치를 수립해야 한다.**

---

### 4.7 특수특성 분류표 (PFMEA)

| FMEA 유형 | 분류 기호 | 명칭 | 기준 | 요구 조치 |
|-----------|---------|------|------|-----------|
| 공정 | **CC** | 중요 특성 (Critical Characteristic) | 심각도 = 9 또는 10 | 특별 공정 관리 필요 (관리 계획서에 명시) |
| 공정 | **SC** | 중요 특성 (Significant Characteristic) | 심각도 = 5~8 **AND** 발생도 = 4~10 | 특별 공정 관리 필요 (관리 계획서에 명시) |
| 공정 | **HI** | 고영향 (High Impact) | 심각도 = 5~8 **AND** 발생도 = 4~11 | 강조 |
| 공정 | **OS** | 작업자 안전 (Operator Safety) | 심각도 = 9~10 | 안전 확인 서명 필요 |
