## 1. 목적

한계 샘플(Limit Sample) 및 표준 샘플(Standard Sample)을 체계적으로 관리하여, 생산·검사 활동에서 일관된 품질 판정 기준을 제공한다.

---

## 2. 적용 범위

M&C Electronics Vietnam 유한회사에서 사용하는 모든 한계 샘플 및 고객으로부터 수령한 표준 샘플에 적용한다.

---

## 3. 회사 발행 한계 샘플 관리 (QM.03.00.15)

**한계 샘플(Limit Sample)**: 개선이 불가능한 불량 유형에 대해, 불합격 경계 수준을 나타내는 현물 샘플. 검사 기준으로 활용한다.

### 3.1 프로세스 흐름

| 단계 | 담당 | 내용 | 기록/양식 |
|------|------|------|-----------|
| **1** | QC / 생산 / R&D | 한계 샘플 신청서를 준비한다. 사진·치수·불량 내용·개선이 불가능한 사유를 포함한다 | - |
| **2** | 담당자 | 불량 내용 및 한계 기준을 검토한다 | - |
| **3** | 담당자 | 고객 제출 필요 여부를 고객 기준에 따라 검토한다 | - |
| **4** | QC 팀장 이상 | 검토 및 승인: 내부 QC가 승인하며, 고객 제출이 필요한 경우 고객 승인도 획득한다. 내용이 부적합한 경우 반려하고 재작성·개선을 요구한다 | - |
| **5** | 담당자 | 관리 목록을 업데이트하고, 사용 유효기간 스티커를 부착한다. 실물과 함께 보관하여 비교 검사가 가능하도록 한다. **유효기간: 6개월 (또는 고객 요건에 따름)** | 관리 목록, 샘플 스티커 |
| **6** | 담당자 | **6개월마다** 샘플 상태를 점검한다 (또는 고객 요건에 따름) | 관리 목록, 샘플 스티커 |
| **7** | 담당자 | **점검 결과**: 계속 사용 가능한 경우 → 유효기간 연장 / 문제 있는 경우 → 샘플 재제작 | 관리 목록, 샘플 스티커 |
| **8** | 담당자 | 관리 목록 업데이트 및 스티커 재부착 | 관리 목록, 샘플 스티커 |

## 4. 고객 제공 표준 샘플 관리 (QM.03.00.14)

**표준 샘플(Standard Sample)**: 고객으로부터 수령하여 검사 기준으로 활용하는 현물 샘플.

### 4.1 프로세스 흐름

| 단계 | 담당 | 내용 | 기록/양식 |
|------|------|------|-----------|
| **1** | 고객 | 고객이 표준 샘플을 발송한다 | - |
| **2** | QC / R&D / 영업 | 고객으로부터 샘플을 수령한다 | - |
| **3** | QC / R&D / 영업 | 샘플에 문제가 없는지 검사한다. 문제 발견 시 고객에게 통보한다 | 이메일, 전화 |
| **4** | 담당자 | 관리 목록을 업데이트하고, 사용 유효기간 스티커를 부착한다. **유효기간 및 검사 기준은 고객 요건에 따른다** | 관리 목록, 샘플 스티커 |
| **5** | 담당자 | **6개월마다** 샘플 상태를 점검한다. 문제 발견 시 고객에게 교체를 요청한다 | 관리 목록, 샘플 스티커, 이메일 |
| **6** | 담당자 | 계속 사용 가능한 경우 → 유효기간 연장 / 문제 있는 경우 → 고객에게 통보하여 교체 진행 | 관리 목록, 샘플 스티커 |
| **7** | 담당자 | 관리 목록 업데이트 및 스티커 재부착 | 관리 목록, 샘플 스티커 |
