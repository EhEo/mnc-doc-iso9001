## 1. 목적

창고 내 모든 자재 및 완제품의 입고·보관·출고 시 **선입선출(FIFO: First In, First Out)** 원칙을 철저히 준수하여 자재의 유효기간 초과, 장기 재고 발생, 품질 저하를 방지한다.

---

## 2. 적용 범위

M&C Electronics Vietnam 유한회사의 모든 창고(원자재 창고, 부자재 창고, 완제품 창고, 도료 창고 등)에 입고·보관·출고되는 모든 자재에 적용한다.

---

## 3. FIFO 원칙 정의

| 용어 | 내용 |
|------|------|
| **선입(First In)** | 먼저 입고된 자재 |
| **선출(First Out)** | 먼저 입고된 자재를 우선 출고 |
| **FIFO 원칙** | 먼저 입고된 자재가 먼저 사용(출고)되도록 관리하는 방식 |

---

## 4. 책임과 권한

| 역할 | 책임 |
|------|------|
| **창고 담당자** | FIFO 라벨 부착, 배치 관리, 출고 순서 준수 |
| **창고 부서장** | 정기 점검, 위반 사항 시정, 교육 실시 |
| **QC 부서** | 감사 시 FIFO 준수 여부 확인 |

---

## 5. FIFO 적용 절차

### 5.1 입고 시

- 새로 입고된 자재는 **기존 재고 뒤쪽 또는 위쪽**에 배치한다.
- 기존 재고는 전면 또는 하단에 위치시켜 먼저 출고될 수 있도록 한다.
- 입고 자재에는 **입고일이 명시된 FIFO 라벨(스탬프)**을 부착한다.

### 5.2 보관 레이아웃 원칙

| 위치 | 배치 자재 | 출고 우선순위 |
|------|-----------|--------------|
| **전면 / 하단** | 먼저 입고된 자재 (구재고) | 우선 출고 ✅ |
| **후면 / 상단** | 나중에 입고된 자재 (신재고) | 후순위 출고 |

### 5.3 출고 시

- 창고 담당자는 출고 요청 처리 시 **반드시 입고일이 빠른 자재부터 출고**한다.
- FIFO 순서에 따르지 않은 출고는 승인 없이 진행 불가.
- 특별한 사유로 FIFO 순서 변경이 필요한 경우, 창고 부서장의 승인 후 진행하고 사유를 기록한다.

## 6. FIFO 식별 표시 (스탬프) 규격

### ⚠️ 2025-07-01부터 적용되는 변경 사항

| 항목 | 구 규격 (2025-06-30 이전) | 신 규격 (2025-07-01부터) |
|------|--------------------------|--------------------------|
| **스탬프 크기** | 80mm × 50mm | **297mm × 210mm (A4)** |
| **부착 방법** | 자재 또는 파렛트에 직접 부착 | A4 용지 크기로 인쇄 후 부착 |
| **적용 시점** | 즉시 폐기 | 2025-07-01부터 신규 스탬프 사용 |

> **참고**: 동일 변경 사항이 창고 관리 프로세스(WH 22.1)에도 적용됨.

### 6.1 FIFO 스탬프 기재 항목 (신규 A4 규격)

- 품명 (Part Name)

- 공급업체명 (Supplier Name)

## 7. 정기 점검

### 7.1 일일 점검

창고 담당자는 매일 순회 점검을 통해 다음을 확인한다:
- FIFO 라벨 부착 여부
- 자재 배치가 FIFO 원칙에 따라 이루어져 있는지 여부
- 유효기간이 임박한 자재 식별 및 우선 출고 조치

### 7.2 주간 점검

창고 부서장은 주 1회 FIFO 준수 여부를 점검하고 결과를 기록한다.

### 7.3 위반 처리

| 위반 유형 | 조치 사항 |
|-----------|-----------|
| FIFO 라벨 미부착 | 즉시 시정, 재발 방지 교육 실시 |
| 배치 순서 위반 | 즉시 재배치, 원인 조사 |
| 출고 순서 위반 | 즉시 시정, 부서장 보고, 재발 방지 대책 수립 |
